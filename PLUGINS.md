the goal of this mechanism is to allow for the separate distribution
of extension binaries 
the 2nd goal is to be able to develop rapidly and reliably extensions with production impact

principle
- using some config file, we can add extra tabs to the GUI,
r/w in the desktop and TUI, r/o in the phone app (monitor)
